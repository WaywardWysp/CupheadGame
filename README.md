# Bayou Brew
  Bayou Brew is a 2D Run&Gun bullet-hell inspired by the game Cuphead, which was developed by Studio MDHR. For our rendition on this classic game, we decided to add 2 new levels based on Mardi Gras aesthetics -- a Circus level and a Bayou level.
