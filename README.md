# Bayou Brew
  Bayou Brew is a 2D Run&Gun bullet-hell inspired by the game Cuphead, which was developed by Studio MDHR. For our rendition on this classic game, we decided to add 2 new levels based on Mardi Gras aesthetics.
Basic Gameplay Requirements : 
- The game has 2 levels, Background music, win and loss music, and a win and loss condition. We also have sound FX on player actions such as jumping, shooting, crouching, etc. The game also has working menu systems, Easy and Hard mode.
